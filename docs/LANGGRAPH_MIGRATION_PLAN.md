# LangGraph 迁移计划

## 📋 项目概述

将现有的 MDT 多智能体系统迁移到 LangGraph 架构，提升工作流的可视化、可维护性和扩展性。

## 🌿 分支策略

### 主要分支
main                           # 生产稳定版本
├── develop                   # 开发集成分支 
│   ├── feature/langgraph-migration # LangGraph 迁移分支 ⭐
│   ├── feature/lg-core       # 核心工作流迁移
│   ├── feature/lg-agents     # 智能体集成
│   └── feature/lg-testing    # 测试和验证
└── hotfix/                   # 紧急修复分支


### 分支保护策略
- `main` 分支：需要 PR + 代码审查
- `develop` 分支：需要 PR + 基础测试
- `feature/langgraph-migration`：允许直接推送，但需要定期同步
